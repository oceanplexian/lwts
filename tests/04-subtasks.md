# Subtasks

## Display on Cards
- Find a card with subtasks (e.g. 3 of 5 done). You should see "3/5" with a progress bar at 60%.
- 1/5 done should show a red bar. 3/5 should be yellow. 5/5 should be green.

## Management in Detail Modal
- Open a card with subtasks. You should see checkbox items for each subtask.
- Type "Write tests" in the add field and click Add. The new subtask should appear in the list.
- Click a checkbox to mark it done. The checkbox should check and the progress bar should update.
- Click the X on a subtask to remove it. It should disappear from the list.
- Drag a subtask to reorder it. The position should change.
