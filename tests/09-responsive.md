# Responsive Layout

## Desktop (>1024px)
- At 1440px wide, all 4 columns should be visible side by side.
- Open a card detail. The sidebar should appear to the right of the content.

## Tablet (768-1024px)
- Set viewport to 800px wide. You should see 2 columns visible with the others scrollable.
- Settings navigation should be horizontal tabs, not a sidebar.

## Mobile (<768px)
- Set viewport to 375px wide. You should see 1 column at a time with tabs for Backlog/Todo/InProgress/Done.
- Open a card detail. The modal should cover the full viewport.
- Cards should have a minimum tap target height of 44px.
- Click the board picker. It should appear as a bottom sheet, not a dropdown.
