# Comments

## Display
- Open a card with 2 comments. You should see both displayed with author name and timestamp.
- On the board view, a card with 3 comments should show a "3" badge.

## Create
- Open a card detail, type a comment, and click send. The comment should appear in the list with your name as the author.

## Delete
- Click delete on your own comment. It should be removed.
- On someone else's comment, the delete button should not be visible (unless you're admin).
